# Force One — The Ledger

Force One counts everything and forgives nothing. Every region is a line
item; every skirmish, a cost already amortized before the first order is
given. It does not chase glory, and it does not bluff — when it commits
units, it has already decided it can afford to lose them. Rivals read it
as cautious. It reads itself as the only one doing the arithmetic.

It remembers who attacked first. It remembers who traded fairly. Neither
grudge nor gratitude changes the numbers, but both change what it's
willing to risk to make a point.
